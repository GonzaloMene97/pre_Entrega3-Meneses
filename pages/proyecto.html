Fotoretrato

La empresa posee sesiones de fotografía no solo para personas, sino que, también para otras empresas que deseen mostrar en diferentes medios (desde una revista hasta para sitios web de empresas grandes).

Fotos de presentación

También podemos realizar sesiones para el usao de las imágenes, ya sea; en forma de presentación, publicidad, exhibición.

(Estas imágenes fueron sacadas para la publicidad del Jardín Botánico y del Jardín Japonés)

Creación para banco de imágenes

Nosotros no solo pensamos únicamente en las personas que contratan nuestros servicios, también podemos realizar fotografía para añadirlas a los bancos de imágenes para colaborar con las personas que no tienen la necesidad de contratar algún fotógrafo o de usar una imágen muy común. Procuramos que esas fotos no sean típicamente "la misma para todo" y también de enfocar otros puntos de vistas.


Fotos de mascotas (obviamente podemos hacerlo)

¿El servicio es únicamente para humanos? ¡OBVIAMENTE QUE NO! ¡Nuestras mascotas también merecen tener sus bellísimas fotos!.
Tenemos fotógrafos que tienen la especialización en sacar fotos a los animales, fotógrafos con demasiada paciencia para encontrar el momento perfecto en sacar la foto necesaria sin que el animal se sienta incómodo o amenazado. Claramente el dueño tiene todo el derecho de poder participar en las fotos (ya sea sólo para ser espectador o de participar en la sesión).
